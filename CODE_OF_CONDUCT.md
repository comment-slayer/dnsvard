# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers.

For now, open a private security-style report as described in `SECURITY.md` and mention it is a Code of Conduct report.
